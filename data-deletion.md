# Data Deletion Instructions for Clara Chat

Effective date: 2026-02-15

## Option 1 (Fastest): Delete in WhatsApp Chat

Send the following command in your Clara Chat conversation:

`!deleteaccount`

Result:

- Your account profile is deleted
- Your stored message history is deleted
- Deletion is permanent

## Option 2: Email Request

If you cannot access WhatsApp, send a deletion request to:

`clarachat@madeofjoy.co`

Please include:

- The phone number used with Clara Chat (E.164 format, e.g. `+447...`)
- A short request text such as "Please delete my Clara Chat data"

We may request verification before processing deletion requests.

## Scope of Deletion

Deletion covers data stored by Clara Chat application systems, including user profile records and message history.

Third-party platform logs (for example Meta/WhatsApp platform records) are governed by those providers' own policies.
